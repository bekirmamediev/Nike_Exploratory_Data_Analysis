Nike Order Completion Analysis

This project analyzes the order item completion rate for Nike products, focusing on products from Nike Official and Nike Vintage. The completion rate is calculated based on delivered items that have not been returned.

Questions

  1. Question 1

    Create a rolling sum that rolls up the number of order items for each product name for the Nike Official business.

    Question 1 SQL
    Techniques Used: CTE (Common Table Expression), Window Function (SUM OVER), INNER JOIN

  2. Question 2

    Calculate the order item completion rate for each product across Nike Official and Nike Vintage.

    Question 2 SQL
    Techniques Used: CTE (Common Table Expression), Conditional Aggregation (SUM CASE WHEN), INNER JOIN, UNION ALL, Division of Aggregates, Ordering

  3. Question 3

    Analyze the order item completion rates per age group.

    Question 3 SQL
    Techniques Used: CTE (Common Table Expression), Conditional Aggregation (SUM CASE WHEN), INNER JOIN, Division of Aggregates, Grouping and Ordering

  4. Question 4

    Calculate the order item completion rate by age group and by age group and product name.

    Question 4 SQL
    Techniques Used: CTE (Common Table Expression), Conditional Aggregation (SUM CASE WHEN), Multiple JOINs, Division of Aggregates, Grouping and Ordering
